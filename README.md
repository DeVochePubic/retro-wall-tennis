# Retro Wall Tennis (テレビ壁打ちテニス)

70年代のビデオゲーム「テレビテニス」や「Pong」にインスパイアされた、2人プレイ専用のレトロな壁打ちテニス（スカッシュ風）ゲームです。

## 特徴
- **スカッシュ風の交互壁打ちルール**: プレイヤー1と2が同じ壁（右側）に向けて交互にボールを打ち返します。自分のターンではないパドルをボールがすり抜けるため、お互いの動きを妨げずスムーズなラリーが楽しめます。
- **70年代ブラウン管 (CRT) シミュレーション**: スキャンライン、画面のちらつき（フリッカー）、残像（ホスファートレイル）、ビネット効果をCSSとCanvasで忠実に再現。
- **Web Audio API によるレトロサウンド**: 外部の音声ファイルを使用せず、オシレーターからリアルタイムにピコピコ効果音を生成。
- **豊富なゲーム設定**: ボール速度調整、勝利スコア変更、カラーテーマ切り替え（CRTグリーン / CRTアンバー / クラシック白黒）。
- **多様な操作方法**: キーボード（1台のPCで2人プレイ）、タッチデバイス、HTML5 Gamepad API（コントローラー対応）。

## プレイ方法（ローカル実行）
このゲームは静的なHTML/CSS/JSのみで構成されているため、ローカルの簡易HTTPサーバー等で起動してブラウザからすぐに遊べます。

### Pythonを使用する場合
ディレクトリ内で以下のコマンドを実行し、ブラウザで `http://localhost:8000` を開きます。
```bash
python3 -m http.server 8000
```

### Node.js (http-server) を使用する場合
```bash
npx http-server
```

## 操作方法
- **Player 1 (RED / Left)**:
  - `W` キー: 上に移動
  - `S` キー: 下に移動
  - タッチ操作: 画面の左端エリアを上下にドラッグ
- **Player 2 (BLUE / Right)**:
  - `↑` キー: 上に移動
  - `↓` キー: 下に移動
  - タッチ操作: 画面の右端エリアを上下にドラッグ
- **共通操作**:
  - `Space` キー: ゲームの一時停止 / 再開
- **ゲームパッド対応**:
  - 接続されたコントローラーのLスティックまたは十字キーでそれぞれのパドルを操作できます。

## 技術スタック

- Vanilla CSS (Glassmorphism, CRT filters, Keyframe animations)
- Vanilla JavaScript (ES6+, Web Audio API, Gamepad API)
